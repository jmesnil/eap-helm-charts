Helm Charts for JBoss EAP Quickstarts

Usage with helm

Helm must be installed to use the charts. Please refer to Helm's documentation to get started.

Once Helm has been set up correctly, add the repo as follows: 

helm repo add jboss-eap-helm-charts https://jmesnil.github.io/eap-helm-charts

If you had already added this repo earlier, run helm repo update to retrieve the latest versions of the packages. You can then run helm search repo jboss-eap-helm-charts to see the charts.

To install the <chart-name> chart: 

helm install my-<chart-name> jboss-eap-helm-charts/<chart-name>

To uninstall the chart: 

helm delete my-<chart-name>

Integration with OpenShift Web Console

The repo can be added to the OpenShift Web Console as a ProjectHelmChartRepository resource to install its charts from the OpenShift Web Console.

The resource definition is: 

apiVersion: helm.openshift.io/v1beta1
kind: ProjectHelmChartRepository
metadata:
  name: jboss-eap-quickstarts-charts
spec:
  connectionConfig:
    url: https://jmesnil.github.io/eap-quickstarts-charts
  name: Quickstarts for JBoss EAP
  description: >-
    This Helm Chart repository contains Helm Charts to deploy JBoss EAP
    quickstarts on OpenShift

You can add it to OpenShift Web Console by pasting it as a new YAML resource or by running the command:

        oc apply -f https://jmesnil.github.io/eap-helm-charts/phcr.yaml
